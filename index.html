Rogueliki

Rogueliki is a roguelike wiki engine.

Goals

Rogueliki provides environments enables users to edit a wiki on roguelike user interface.
Users are able to use Rogueliki casually, using portable data, cross-platform developments and others.

Methods

Rogueliki is developed in JavaScript, works on IE 6.0, Firefox 2.0 and so on.
